Get a network within GCE from its name.

{{% examples %}}
## Example Usage
{{% example %}}

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as gcp from "@pulumi/gcp";

const my_network = gcp.compute.getNetwork({
    name: "default-us-east1",
});
```
